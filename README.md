# Software Engineering - Projects

A custom experimentation and learning project to explore real-world use cases
using best practices and design patterns to solve real-world problems.

## Projects

| Project | Description | Status |
|---|---|---|
| [Distributed Order System](impl/distributed-order-system/) | A Go-based microservices system demonstrating distributed systems patterns such as choreography saga, CQRS, and competing consumers using RabbitMQ, PostgreSQL, and Redis | Done |
| [Performance Cost Optimization](impl/performance-cost-optimization/) | A project exploring Cache-Aside, Compute Resource Consolidation, and Backend For Frontends patterns with focus on latency, freshness, resource utilization, and scaling boundaries | Done |
| [Resilient API Gateway Layer](impl/resilient-api-gateway-layer/) | A Go-based API gateway implementing Ambassador, Circuit Breaker, Bulkhead, and Retry with backoff patterns with unit and integration tests using httptest fake servers to validate resilience behavior under failure conditions | Done |
