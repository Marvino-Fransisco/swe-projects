# Software Engineering

A custom experimentation and learning project to explore real-world use cases
using best practices and design patterns to solve real-world problems.

## Projects

| Project | Description |
|---|---|
| [Distributed Order System](impl/distributed-order-system/) | A Go-based microservices system demonstrating distributed systems patterns such as choreography saga, CQRS, and competing consumers using RabbitMQ, PostgreSQL, and Redis |
| [Performance Cost Optimization](impl/performance-cost-optimization/) | A project exploring Cache-Aside, Compute Resource Consolidation, and Backend For Frontends patterns with focus on latency, freshness, resource utilization, and scaling boundaries |
| [Resilient API Gateway Layer](impl/resilient-api-gateway-layer/) | A project implementing Ambassador, Circuit Breaker, Retry, and Bulkhead patterns to handle downstream service failures, timeouts, and retry vs fail-fast decisions |
